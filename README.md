# nyanplot
本ライブラリはグラフ作成用のライブラリです。

## install手順

#仮想環境の作成
>conda create -n <env_name>

>conda activate <env_name>


>git clone https://github.com/mykoyan/nyanplot.git

>cd nyanplot

#依存環境のインストール

>pip install -r requirements.txt

#nyanplotのインストール

>pip install .

#jupyterのインストール

>conda install activate jupyter
